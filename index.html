How the Internet works

Accessing a web page, such as Expedia.com, results in a HTTP requestsent to servers for page information. Page information on the server is stored as an HTML document. When HTML is sent back to users computer, the web browser displays information in a readable format

Hypertext Markup Language (HTML)

HTML includes TEXT (“what you see”) and MARKUP (“how it looks”). HTML documents are made up of:

1. Text Content – What you see

2. Markup - What it looks like

3. References to other documents

4. Links to other pages


HTML Tags and Elements

HTML is made up of HTML tags and elements. Tags will typically have an opening tag and a closing tag. If elements have no content, then no closing tag is needed (“void tag”). Elements are made up of inline or block elements. Block elements create an invisible block of content.

Basic HTML tags include:

b = bold

a = links

img = Image

br = break

p = paragraph
